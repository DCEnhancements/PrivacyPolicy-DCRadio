# PrivacyPolicy-DCRadio

## Information Collection and Use

DC Enhancements built the DC Radio app as a mobile application. This SERVICE is provided by DC Enhancements and is intended for use as is.
The app may process technical or account-related data required to deliver its features, keep the service secure, and resolve support issues. We limit collection to the minimum necessary for the feature you actively use.

## Permissions

The app may request the following Android permissions to provide its features:
- `android.permission.INTERNET` — connect to online services that provide core app functionality
- `android.permission.ACCESS_NETWORK_STATE` — detect whether a network connection is available before contacting remote services
- `android.permission.FOREGROUND_SERVICE` — keep long-running playback or sync tasks active while the app is in use
- `android.permission.FOREGROUND_SERVICE_MEDIA_PLAYBACK` — support a user-facing feature that depends on this Android permission
- `android.permission.POST_NOTIFICATIONS` — deliver playback, status, or product notifications

## Third-Party Services

The app integrates the following third-party services or SDK categories:
- **AudD** — This service may process technical request metadata necessary to provide its feature.
- **Retrofit / HTTP APIs** — This service may process technical request metadata necessary to provide its feature.

## Ads and In-App Purchases

The app does not include advertising and does not process in-app purchases unless this policy is updated to state otherwise.

## Data Retention
